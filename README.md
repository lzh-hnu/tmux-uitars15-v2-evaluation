# tmux for UITARS15_v2: Comparative Terminal Evaluation

A focused benchmark discussion for terminal-action grounding and ablation design.

This repository is a public GitHub Pages research discussion site for tmux adaptation on **UITARS15_v2**.

## Pages

The published site is expected at:

https://lzh-hnu.github.io/tmux-uitars15-v2-evaluation/

## Structure

- `docs/index.html` - academic discussion page with the project image at the top
- `docs/assets/hero.png` - generated 700x500 project image
- `docs/research-notes.md` - research framing, evaluation lens, and validity notes
- `docs/styles.css` - static page styling

## Scope

This is a discussion-oriented scaffold, not a full implementation of the tmux adapter. It is prepared for later expansion with executable harnesses, trace schemas, and benchmark scripts.
